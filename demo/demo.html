DomOpsParty

Manages the component hierarchy. Each branch owns its DOM elements directly via branch.createElement(). Calling dissolve() removes the branch, all its elements, and all sub-branches in one call — no manual cleanup needed.

⚠ LeakyComponent — Bad Example

This component creates a branch and elements but never calls dissolve() before going out of scope. The branch node and its elements are permanently orphaned in the party tree.

Leak Scanner

Periodically walks the branch tree checking isOwnerAlive on every branch. When an owner has been GC'd the branch is highlighted red in the tree and auto-dissolved.
